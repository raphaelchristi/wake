# Code of Conduct

This project adopts the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. Please read the full text at the link above before contributing.

## Our standards in one paragraph

Be respectful. Assume good intent. Disagree with ideas, not people. Help newcomers. Don't tolerate abuse. Keep technical disagreements technical.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainers at:

> **valdetaroraphael@gmail.com**

All complaints will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter of any incident.

Maintainers who do not follow or enforce the Code of Conduct in good faith may face temporary or permanent repercussions as determined by other members of the project's leadership.

## Scope

This Code of Conduct applies within all project spaces (GitHub issues, pull requests, discussions, Discord when it launches), and also when an individual is representing the project in public spaces.
